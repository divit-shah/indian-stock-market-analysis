# Indian Stock Market Analysis 2024

Comparative analysis of TCS, Reliance Industries, HDFC Bank, and Infosys 
stock performance throughout 2024, including price trends, cumulative 
returns, and daily return distribution.

## Tools Used
- Python
- yfinance
- pandas
- matplotlib

## Key Findings
- Reliance delivered the strongest returns in 2024 (~27%)
- TCS recovered from an 18% drawdown in Q1 to end the year positive
- All four stocks dropped together in October 2024 due to FII outflows
- TCS daily returns approximately follow a normal distribution
